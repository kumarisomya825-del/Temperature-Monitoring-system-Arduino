# Temperature Monitoring System using Arduino

This project reads temperature from DHT11 sensor and displays it on 16x2 LCD.

## Components used:
- Arduino UNO
- DHT11 Sensor
- 16x2 LCD
- Potentiometer
- Breadboard
- Jumper wires

## Output:
Temperature is displayed on LCD in real-time.
